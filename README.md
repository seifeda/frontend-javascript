# 0x04-TypeScript

## Overview

This repository contains **TypeScript exercises** focusing on fundamental and advanced TypeScript concepts.  
By completing these tasks, you will learn:

- Basic types in TypeScript
- Interfaces, classes, and functions
- Working with the DOM in TypeScript
- Generic types
- Namespaces and declaration merging
- Ambient namespaces and importing external libraries
- Basic nominal typing

Each task is self-contained and demonstrates one or more TypeScript concepts in a practical way.
